Parágrafos e quebras de linha

Podemos escrever parágrafos de qualquer jeito que quisermos. Basta colocar tudo no meio do par de tags <p> e </p>.


Importante salientar que HTML é uma linguagem de marcação, portanto, pouco importa a quantidade de quebra de linhas que se faça no editor, o documento final isso não se traduzirá em quebras de linhas.
Para que ocorra uma quebra de linha, é necessário usar a tag <br>.

Vamos adicionar alguns simbolos especiais:
®: Rights Reserved
©: Comercial
™: TM
€: Euro
£: Libra
¥: Yen
¢: Centavos de Dólar
δ: delta minúsculo
Δ: delta maiúsculo
↑: Seta Acima
